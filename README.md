# ecommerce-analysis
🚀 Completed My First End-to-End E-Commerce Data Analysis Project using Power BI

I worked on an E-commerce dataset and built 3 interactive dashboards focused on:

📊 Sales Performance Overview
👥 Customer Insights & Retention
📦 Product & Category Performance

# 🔹 Data Preparation

Before building the dashboards, I worked on:

* Cleaning and transforming data
* Fixing data types and validating data integrity
* Building relationships between tables
* Removing unused columns
* Handling duplicates and missing values
* Creating DAX Measures & Calculated Columns
* Preparing KPIs and metrics for analysis

# 🔹 Dashboard 1: Sales Performance Overview

This dashboard helped identify:

* Strong seasonal trends with clear sales peaks and low periods
* A small group of products contributing the majority of revenue, with some products generating over 10M in sales
* Low-performing products creating storage costs with minimal return
* High-performing and low-performing cities through geographic sales analysis

These insights can support:

* Better seasonal planning
* Inventory optimization
* Regional marketing strategies
* Distribution and shipping improvements

# 🔹 Dashboard 2: Customer Insights & Retention

Key findings:

* Consistent customer growth from 2023 to 2025
* Returning customers represent the majority of total customers
* Average Orders per Customer = 4.39
* Repeat Purchase Rate = 83%

The analysis also revealed that:

* High-frequency buyers represent the largest and most valuable customer segment
* Customer retention is significantly more valuable than acquiring new customers
* Top customers generate exceptionally high Customer Lifetime Value (CLV), making retention strategies critical

# 🔹 Dashboard 3: Product & Category Performance

Main insights:

* Very high Inventory Turnover Rate indicates fast-moving inventory with relatively low stock levels
* Higher-priced products are still achieving strong sales performance
* Stock distribution across products is not fully balanced with actual demand
* Scatter plot analysis showed a non-linear relationship between price and quantity sold, indicating varying price elasticity across products
